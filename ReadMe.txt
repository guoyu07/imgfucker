这么简单个东西就不用社么框架之类的了

操作流程：
	1.去贴吧发帖子，然后添加一个一像素的图片。
	2.后台可以通过别人打开帖子打开我们图片的时候，获取到这个广告位所在的位子（url）。
	3.记录这个广告的位子，并记录最后20个浏览的人的IP
	4.到了半夜时候就在后台将这个一像素的图片替换成我们的广告。
	5.在后台运行一个cron脚本，查看每一张帖子最后被打开的时间，如果在正常时间（普通人上网时间）出现没流量情况，那就记录他为被封了，
	6.找出同一板块被封的帖子，然后对他们的IP取交集，找出管理员IP


记录最后20个来访者的IP和UserAgent（计数器统计），循环使用
记录格式：     LogofUser.{site}.{form}.{tid}.{num} = {ip}
计数格式：     LogofUserCurrentNum.{site}.{form}.{tid} = {num}
逻辑：先根据计数来取得当前的id
